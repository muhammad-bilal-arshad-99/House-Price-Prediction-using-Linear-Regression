# House-Price-Prediction-using-Linear-Regression
A beginner-friendly machine learning project that predicts house prices using Linear Regression based on area, number of bedrooms, and number of bathrooms.
About this Project:

Machine learning was a field I always wanted to explore, and this was one of my first complete projects. I built this house price prediction model in Python to understand how a machine learning algorithm learns from data and makes predictions.

The model estimates the price of a house using three basic features:

Area

Number of bedrooms

Number of bathrooms

Although the project is simple, it gave me hands-on experience with the complete machine learning workflow—from preparing data to training a regression model and making predictions through a console application.
